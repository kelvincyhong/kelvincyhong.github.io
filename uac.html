Universal Accessibility for Confluence

// February 2018

As part of a ShipIt project at Atlassian, I worked on designing the user interface for Accessibility Mode on Confluence as a hackathon project.


The Accessibility mode included a low and high contrast mode for those with visual disabilities, a dyslexic function to aid those with dyslexia and a seamless zoom mode.


While designing these mockups, I did research and took into account the Web Content Accessibility Guidelines (WCAG) 2.0. Other design principles used included Color Theory, Typography, Layout and User Experience.
